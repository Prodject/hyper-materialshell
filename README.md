### Install

1. Open HyperTerm's preferences with your editor - Tip: `CMD+,` or at `~/.hyperterm.js`.
2. Add `hyperterm-materialshell` to your plugins list:

```js
plugins: ['hyperterm-materialshell']
```

3. Reload HyperTerm - `CMD+Shift+R`.
